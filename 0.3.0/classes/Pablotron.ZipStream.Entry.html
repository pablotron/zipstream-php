Methods

Check file path.

check_path(string $path) : void

Verifies that the given file path satisfies the following constraints:

  • Path is not null.
  • Path not empty.
  • Path is less than 65535 bytes in length.
  • Path does not contain a leading slash.
  • Path does not contain a trailing slash.
  • Path does not contain double slashes.
  • Path does not contain backslashes.
  • Path is not a relative path.
Throws
\Pablotron\ZipStream\PathError

if path is invalid.

Arguments

$path

string

Input file path.
